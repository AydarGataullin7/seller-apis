# Файл ````seller.py````

## Что делает
Этот файл автоматически обновляет информацию о товарах в интернет-магазине на платформе Ozon и следит за актуальными ценами и количеством товаров на складе.

## Какие задачи выполняет
- Автоматическое обновление данных: регулярно проверяет и обновляет информацию о товарах  
- Синхронизация цен: следит за тем, чтобы цены на Ozon соответствовали актуальным  
- Контроль остатков: автоматически обновляет информацию о количестве доступных товаров  
- Автоматическое добавление: если появляются новые товары, программа добавляет их в каталог

## Откуда берутся данные
Все данные берутся из специального файла, который скачивается с сайта поставщика ````(https://timeworld.ru)````. Этот файл содержит актуальную информацию о ценах и наличии товаров.
