# Файл ````seller.py````

## Что делает
Этот файл автоматически обновляет информацию о товарах в интернет-магазине на платформе Ozon и следит за актуальными ценами и количеством товаров на складе.

## Какие задачи выполняет
- Автоматическое обновление данных: регулярно проверяет и обновляет информацию о товарах  
- Синхронизация цен: следит за тем, чтобы цены на Ozon соответствовали актуальным  
- Контроль остатков: автоматически обновляет информацию о количестве доступных товаров  
- Автоматическое добавление: если появляются новые товары, программа добавляет их в каталог

## Откуда берутся данные
Все данные берутся из специального файла, который скачивается с сайта поставщика ````(https://timeworld.ru)````. Этот файл содержит актуальную информацию о ценах и наличии товаров.

# Файл ````market.py````

## Что делает
Этот файл автоматически синхронизирует информацию о товарах между сайтом поставщика и маркетплейсом Яндекс.Маркет. Он поддерживает две разные схемы работы магазина на платформе.

## Какие задачи выполняет
- Двойная синхронизация: работает одновременно с двумя типами магазинов на Яндекс.Маркете
- Обновление остатков: автоматически обновляет информацию о количестве товаров на складах
- Синхронизация цен: обеспечивает соответствие цен на Маркете актуальным ценам поставщика
- Поддержка разных схем: работает как с товарами, которые хранятся у продавца, так и с теми, которые хранятся на складах Яндекса

## Откуда берутся данные
Все данные берутся из того же файла поставщика (https://timeworld.ru), что и для Ozon. Программа использует общий источник данных для обеих платформ.
